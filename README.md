Proyek ini mengembangkan chatbot berbasis Large Language Model (LLM) yang di-fine-tune menggunakan teknik **LoRA (Low-Rank Adaptation)** untuk menjawab pertanyaan seputar **kuliner dan sejarah Yogyakarta**.
